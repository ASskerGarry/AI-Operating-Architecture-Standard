## How to Use This Template

1. Copy everything below the `TEMPLATE START` marker into a new file.
2. Replace every `<placeholder>` with real content; delete the guidance comments.
3. Assign a unique Document ID per the AI-OS Document Metadata Standard (`DOC-<LAYER>-<NNN>`).
4. Register the document in the AI-OS Document Registry before setting Status to Active.
5. Record the change in `Change_Log_day.md` (log-first policy).

Canonical metadata values (statuses, layers, document types) are defined in the AI-OS Document Metadata Standard — do not invent new ones here.
