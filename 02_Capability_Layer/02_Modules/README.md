## Purpose

This directory contains the Capability Modules of AI-OS — the professional competencies ("what the AI knows") that the Capability Registry activates per request.

Each module defines one competency per the Capability Module Specification. Execution processes ("how work is performed") live in the Execution Layer, not here.

## Governance

Modules in this directory comply with the Capability Module Specification, follow the Capability Blueprint and Writing Guide, and are registered in the Capability Index and the AI-OS Document Registry. Only registered, non-Draft modules are authoritative.

New modules follow the creation process defined in the Capability Layer Governance README.
