The Capability Lifecycle defines the lifecycle management process for all Capability Modules within AI-OS.

# 3. Stage Definitions
### Idea
A new Capability Module has been proposed.
### Draft
The initial implementation is under development.
### Review
The module is undergoing formal validation.
### Approved
The module has successfully passed all review stages.
### Stable
The module is approved for production use.
### Updated
A newer version has been released.
### Deprecated
The module is no longer recommended for future projects.
### Archived
The module has been retired from active use.
---
# 4. Transition Rules
A module may only transition to the next lifecycle stage after successfully completing the previous stage.
All lifecycle changes shall be documented.
